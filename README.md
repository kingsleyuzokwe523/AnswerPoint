# AnswerPoint - WAEC & NECO Answers Portal

A simple web application for managing and displaying exam answers using PIN codes.

## Features
- 3-digit PIN system for answers
- Support for WAEC and NECO subjects (80+ subjects)
- Admin dashboard with analytics
- View counter for each PIN/subject
- Image upload for answers
- Exam timetable display
- Editable home page content
